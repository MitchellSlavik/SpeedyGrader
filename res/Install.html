Install Instructions

This instruction manual is for using SpeedyGrader on a windows machine.
All other operating systems are untested and I am leaving it up to you
to figure it out if you choose to use this software on anything other
than windows. >:D


Java

Before starting these steps make sure typing "javac -version" in a
command prompt does not give any version information or doing
this will be pointless.


You obviously already have the java JRE installed on your computer just
to read this manual, but to use SpeedyGrader to grade java programs you
must have the java JDK installed to your computer as well. You can find
the link for the JDK at the bottom of this page. Just click on the
"Java Platform (JDK)" and download/install it.


After it is installed, you must set your classpath on your computer to
point to the newly installed JDK. To do this you navigate to your
"Advanced System Settings", click "Environmental Variables...", then
click "edit" with your System Variable "Path" selected. Now go find the
path to the bin folder of the JDK (It should look something like
"C:\Program Files\Java\jdk#.#.#_##\bin"). Append that path to your
"Path" System Variable (Windows 7 and earlier, add a ';' before
appending the JDK bin path).


If everything went OK then you should be able to open a command prompt,
type "javac -version" and get version output. Congrats you are now
set to go to use SpeedyGrader for grading java programs.
SpeedyGrader will require a restart to work after following these steps.


C++

Before starting these steps make sure typing "g++ --version" in a
command prompt does not give any version information or doing
this will be pointless.


To start grading C++ files with SpeedyGrader, you must first install
MinGW. Download MinGW from the link at the bottom of this page. After
opening the installer, click "Install". Make a note of the installation
directory (Usually "C:\MinGW"), then click "Continue". Wait for the
installations to finish downloading and installing.


After the MinGW installer finishes, it should open a new window that
allows you the choose MinGW packages. Click the checkbox next the
package that ends in "g++". Then go to the top and select
"Installation > Apply changes". Another window will appear, hit
"Apply". Wait for more downloads/installations to complete. After
the installations are complete, close the "MinGW Installation Manager"
window.


After it is installed, you must set your classpath on your computer to
point to the newly installed MinGW bin. To do this you navigate to your
"Advanced System Settings", click "Environmental Variables...", then
click "edit" with your System Variable "Path" selected. Now go find the
path that MinGW installed to and get the path of the bin folder
(It should look something like "C:\MinGW\bin"). Append that path to your
"Path" System Variable (Windows 7 and earlier, add a ';' before
appending the MinGW bin path).


If everything went OK then you should be able to open a command prompt,
type "g++ --version" and get version output. Congrats you are now set to
go to use SpeedyGrader for grading C++ programs.
SpeedyGrader will require a restart to work after following these steps.
